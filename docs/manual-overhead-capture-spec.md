# Manual overhead capture handoff

Capture against the current 2026/current-build map geometry. Use a true near-orthographic top-down camera, with consistent camera roll and orientation across every frame. Do not include the HUD, player markers, or other gameplay overlays.

- Capture each frame at 1920×1080 minimum.
- For multi-frame routes, maintain 20–30% overlap between adjacent frames.
- Retain the original captures; do not keep only a stitched or optimized derivative.
- Control maps require all three stages, clearly identified in the handoff.
- Record capture date, game build, map, and stage for every capture.
