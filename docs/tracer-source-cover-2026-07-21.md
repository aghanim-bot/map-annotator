# Tracer source acquisition cover — 2026-07-21

Research cutoff: **2026-07-21**. This is an offline metadata and caption audit, not a visual annotation audit.

## Result

The 13 metadata leads were all retrieved successfully and all 39 supplied artifacts passed the parent checksum manifest before parsing. All 13 sources are genuinely educational or coaching-oriented and retain title-level historical map identity. They cover **10 maps**. The caption audit promotes **30 exact spoken contexts** (three per covered map) to a map-specific, **pending-frame** shortlist.

These are not visually verified points or routes. A title can establish which map a map-wide guide discusses, but it cannot establish current map geometry, the visible object, route endpoints, coordinates, objective phase, or side at a timestamp. Sources from 2017–2021 predate Overwatch 2 and are discovery evidence for route concepts only; they cannot establish current geometry. The 2023 Shambali Monastery and Paraíso sources are newer, but remain dated and still require current-version regression review. Every shortlisted row therefore has unresolved objective phase and requires a narrowly bounded, checksummed high-resolution current-version frame/video review, compared against the dated 2026 corpus, before promotion to annotation. No claim below is based on visual inspection.

## Follow-up gate

For any row promoted beyond this shortlist, retrieve only a narrow high-resolution current-version window around the timestamp, save and checksum it independently, and review continuous frames. Compare those current-version frames against the dated 2026 corpus before promotion. The review must establish readable/unambiguous current map identity plus the exact claimed object or route start, direction, and endpoint; title-based map identity alone is insufficient evidence of current geometry. Objective phase and coordinates remain unresolved unless explicit source text or the later verified frames establish them.

The older unslotted long-form source remains governed by `docs/tracer-source-segmentation-2026-07-21.md`: it contributes no map slot and is not included in the counts above.
